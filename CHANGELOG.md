# Changelog

All notable changes to `filament-edit-page-tester` will be documented in this file.

## 1.0.0 - 2026-01-21

### Added
- Initial release
- Support for all standard Filament v4 field types
- Automated form filling
- Field preview testing
- Save validation
- Custom field handler support via interface
- Verbose logging mode
- Display value mapping
- Field callbacks for custom handling
- Comprehensive documentation and examples
- MIT License

### Supported Field Types
- TextInput
- Textarea
- Select
- RichEditor
- DatePicker
- DateTimePicker
- TimePicker (comparison only)
- Toggle
- Checkbox
- CheckboxList
- CodeEditor
- KeyValue

### Features
- Fluent API for readable tests
- Required field validation
- Custom field handlers for application-specific fields
- Extensible architecture
- Compatible with Pest v3 and v4
- Full Filament v4 support
